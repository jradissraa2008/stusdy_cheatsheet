Idée générale — Approximation
Principe
On calcule terme par terme # On s'arrête quand |V2 - V1| < epsilon # V1 = terme précédent, V2 = terme courant
Structure type
sig = -1 # signe alterné v2 = 1 # valeur courante v1 = 0 # valeur précédente i = 2 # exposant départ (pair) while abs(v2 - v1) >= epsilon: v1 = v2 v2 = v2 + terme(x,i) * sig i += 2 sig = -sig
Exercice 1 — Cos(x)
Formule
Cos(x) = 1 - x²/2! + x⁴/4! - x⁶/6! + … # terme(i) = x^i / fact(i) * sig
Code Python
def calcul_cos(x): sig=-1; v2=1; i=2; v1=0 while abs(v2-v1) >= 0.0001: v1 = v2 v2 = v2 + (puis(x,i)/fact(i))*sig i += 2; sig = -sig print("cos(",x,")=",v1)
Fonctions auxiliaires
def puis(a,b): # a^b p=1 for i in range(1,b+1): p=p*a return p def fact(x): # x! f=1 for i in range(1,x+1): f=f*i return f
Exercice 2 — Constante K
Formule
K = Σ (-1)ⁿ / (2n+1)² = 1 - 1/9 + 1/25 - 1/49 + … # terme(n) = 1/(i*i), i = 1,3,5,7,... # epsilon : 0.01 ≤ eps ≤ 0.00001
Saisie epsilon
def saisie(): eps=float(input("eps=")) while not (0.00001<=eps<=0.01): eps=float(input("eps=")) return eps
Calcul K
def calcul_k(eps): v2=1; v1=0 sig=-1; i=3 while abs(v2-v1)>=eps: v1=v2 v2=v2+(1/(i*i))*sig sig=-sig i+=2 print("K=",v1)

Exercice Cryptage — Principe
Fonctionnement
1. Saisir MOT (majuscules, ≤25 chars)
2. Tableau T aléatoire (1..25), taille = len(MOT)
3. Remplacer chaque lettre par +p positions
4. Si dépassé 'Z' → revenir à 'A' (modulo)
5. Trier la chaine résultat alphabétiquement
Exemple SALUT
Lettre+pRésultat
S+19L
A+3D
L+7S
U+18M
T+15I
LDSMI → trié → DILMS
Fonction crypter
def crypter(ch): n=len(ch) t=array([int()]*n) for i in range(n): t[i]=randint(1,25) ch1="" for i in range(n): if ord(ch[i])+t[i]>90: ch1+=chr(ord(ch[i])+t[i]-26) else: ch1+=chr(ord(ch[i])+t[i]) ch1=tri(ch1) return ch1
ASCII à retenir
'A' = 65 | 'Z' = 90 ord('A') = 65 chr(65) = 'A' Si ord(lettre)+p > 90 → -26 (retour A)
